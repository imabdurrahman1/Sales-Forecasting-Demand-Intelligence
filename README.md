# 📈 End-to-End Sales Forecasting & Demand Intelligence System

## Overview

This project was developed as part of an AI & Data Science Internship.

The objective is to forecast future product sales, detect unusual sales patterns, segment products based on demand, and provide business insights through an interactive Streamlit dashboard.

---

## Features

- Sales Trend Analysis
- Time Series Decomposition
- SARIMA Forecasting
- Facebook Prophet Forecasting
- XGBoost Forecasting
- Model Comparison
- Isolation Forest Anomaly Detection
- Rolling Z-Score Detection
- Product Demand Segmentation using K-Means
- Interactive Streamlit Dashboard

---

## Technologies

- Python
- Pandas
